# nnwm

**no name Window Manager (nnWM)** is a lightweight and minimalistic tiling wayland compositor.

Latest Version: 0.1.0

# Features

- Tiling window management
- Configuration through Lua
- Based on `wlroots` (similar to SwayWM/RiverWM)
